Overview

Purpose

The purpose of this website is to assist in advertising and organizing the company. In this website will display different services we can perform and how those services will improve their overall security. The first page will be an overview of the website and everything we do. The second page will be an informational page that explains what a penetration test is and the different types of penetration testers. This page is important because companies need to understand what penetration tests are and how they work before in order for them to see why they need penetration tests. Page 3 will be a place we can add different atricles that talk about different aspects of security. This can be beneficial for companies to be up to date with different security trends.

Audience

The intended audiences are companies in need of security professionals to assist in securing their company. Companies can contact us and ask for a penetration test to be done in their systems. This company can also provide information for anyone who wants to learn about the security trends. We have a blog tab that will bring the user multiple articles that talk about security concepts. These articles can help companies stay up to date and not miss out on any vital pieces of information regarding security.
